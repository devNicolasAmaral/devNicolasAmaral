Aqui estão algumas soluções que desenvolvi:

#### 1. 🤖 Bot de Extração de Faturas (Neoenergia/Utilities)
* **O Problema:** Necessidade de baixar manualmente dezenas de faturas em PDF de um portal com login e CAPTCHA.
